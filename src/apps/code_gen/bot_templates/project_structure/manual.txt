Инструкция по запуску чат-бота с сайта https://freebots.ru

Следуйте шагам в инструкции, для успешной настройки и запуска чат-бота.

Более наглядная инструкция по ссылке: https://freebots.ru/manual

1) Настройка конфига чат-бота
Конфиг - это файл .env, который содержит необходимые для запуска чат-бота данные.

Выполните следующие действия для настройки конфига:

1.1) Создайте отдельную папку для чат-бота и перенесите в нее все файлы из ZIP-архива
1.2) Переименуйте файл .env.example в .env
1.3) В файле .env укажите токен чат-бота (получите его в Telegram-боте @BotFather) и ваш идентификатор в Telegram
(получите его в Telegram-боте @getmyid_bot)
1.4) Если вы используете отправку писем на электронную почту, необходимо указать вашу Gmail почту и пароль в файле .env,
а также пароль приложения Google (как получить пароль приложения Google читайте по ссылке: https://support.google.com/mail/answer/185833?hl=ru


2) Запуск чат-бота

2.1) Запуск с помощью Docker
Если у вас установлен Docker, то с помощью него вы легко можете запустить чат-бота. В случае, если у вас нет Docker,
перейдите к разделу "2.2) Настройка виртуального окружения, установка пакетов и запуск".

Для запуска с помощью Docker выполните следующие действия:
2.1.1) Создайте образ с помощью команды docker build . -t freebot
2.1.2) Запустите Docker-контейнера с помощью команды docker run -d freebot

2.2) Настройка виртуального окружения, установка пакетов и запуск вручную
Если у вас не был установлен Docker, то вы можете настроить и запустить чат-бота вручную. Выполните следующие действия
для успешной настройки виртуального окружения, установки пакетов и запуска:

2.2.1) Установите Python версии 3.10 или выше с официального сайта: https://www.python.org/downloads/release/python-3124/
(при установке ставьте галочки на все пункты).

2.2.2) В папке с чат-ботом откройте командную строку и в зависимости от ОС выполните необходимые команды.
Команды для Windows:
- Создайте виртуальное окружение с помощью команды python -m venv venv
- Активируйте виртуальное окружение с помощью команды venv\Scripts\activate
- Установите пакеты с помощью команды pip install -r requirements.txt
- Запустите чат-бота с помощью команды python main.py

Команды для Linux и macOS:
- Создайте виртуальное окружение с помощью команды python3 -m venv venv
- Активируйте виртуальное окружение с помощью команды source venv/bin/activate
- Установите пакеты с помощью команды pip install -r requirements.txt
- Запустите чат-бота с помощью команды python3 main.py
- Если вы закрыли командную строку и хотите снова запустить чат-бота, вам понадобиться активировать виртуальное
окружение в той папке, где вы его создали, но создавать его снова и скачивать пакеты больше не нужно.
